## エグゼクティブサマリー

- **🚀 Claude Sonnet 5 が GA、Claude Code のデフォルトモデルが Opus 4.8 → Sonnet 5 に転換**（6/30・v2.1.197）。1M トークンコンテキストをデフォルト搭載し、導入価格 $2/$10 per MTok（8/31 まで）。5/28 以来続いた「Opus デフォルト」を覆すコスト効率重視の大転換。
- **🌐 Claude in Chrome が一般提供（GA）に到達**（7/1・v2.1.198）。ターミナルでビルド→ブラウザで検証・デバッグする Claude Code 連携が正式提供。
- **🚨 Claude Fable 5 / Mythos 5 のアクセスが復旧**（7/1）。米商務省が 6/30 夜に輸出管理規制を解除し、公開版 Fable 5 がグローバル復旧、Mythos 5 は承認済み米国機関限定で復旧。6/12 以来の全面停止が解除された。
- **🏢 Claude が Microsoft Foundry で GA**（6/29）。Bedrock / Vertex と並ぶエンタープライズ・プロバイダとして整備が進行。
- **💼 6/15 のサブスク課金構造変更（programmatic 利用の別建てクレジット化）は「施行されず一時停止」だったことが判明・訂正**。加えて Anthropic が Alibaba を蒸留目的の不正アクセスで告発（Bloomberg）。

期間中は CLI が **v2.1.190 → v2.1.198** まで進行（機能追加 5 本＋安定化・欠番）。モデル戦略（Sonnet 5 デフォルト化）・ブラウザ連携（Chrome GA）・自律コード納品（BG エージェント自動 PR）と、ユーザー体験を左右する大型トピックが集中した週。並行して Fable 5 / Mythos 5 の輸出規制解除という地政学・ビジネス面の決着も見られ、全体トーンは「メジャー機能の一斉着地」。

## 🚀 最新機能系

### Claude Sonnet 5 GA ＋ Claude Code のデフォルトモデル切替 ✅ GA【最重要】
- **概要**: Sonnet 系次世代モデル `claude-sonnet-5` が GA。**1M トークンコンテキストがデフォルト（かつ最大）**、最大出力 128k。Claude Code のデフォルトが **Opus 4.8 → Sonnet 5** に変更（要 v2.1.197+、Opus 4.8 は `/model` で選択継続）。SWE-bench Pro 63.2%（Sonnet 首位）・GDPval-AA v2 で Opus 4.8 をわずかに超え。⚠️ adaptive thinking デフォルト ON／手動 extended thinking 廃止（400）／sampling パラメータ非デフォルト値 400／新トークナイザで同一テキスト約 30% 増、といった挙動変更に注意。
- **対象**: Free / Pro のデフォルト、Max / Team / Enterprise でも利用可。導入価格 $2/$10 per MTok（8/31 まで）→ 標準 $3/$15。
- **初出**: v2.1.197 (2026-06-30) / モデルは 2026-06-30 GA
- **情報源**: [日次レポート 7/1](../reports/2026-07-01_v2.1.197-claude-sonnet-5-default-model.md) / [Introducing Claude Sonnet 5](https://www.anthropic.com/news/claude-sonnet-5)

### Claude in Chrome が一般提供（GA）✅ GA
- **概要**: ブラウザサイドパネルで Claude が Web ページを読む・クリック・ナビゲートする Chrome 拡張が GA。**Claude Code 連携**でターミナルのビルド→ブラウザ検証、コンソールエラー・DOM 状態を直接読んでデバッグが可能。プロンプトインジェクション防御を強化（攻撃成功率 23.6%→11.2%、ブラウザ特化型は 0%）。
- **対象**: Pro / Max / Team / Enterprise（有料）。Team/Enterprise は組織横断で有効化・サイト allow/blocklist 設定可。
- **初出**: v2.1.198 (2026-07-01) changelog 筆頭で GA 告知
- **情報源**: [日次レポート 7/2](../reports/2026-07-02_v2.1.198-chrome-ga-and-fable5-restored.md) / [Claude for Chrome](https://claude.com/claude-for-chrome)

### バックグラウンドエージェントの自動 commit / push / ドラフト PR ✅ GA
- **概要**: `claude agents` から起動した BG エージェントが worktree 内でコード作業を完了した際、停止して確認を求める代わりに**自動で commit・push し、ドラフト PR を作成**。完全自律のコード納品フローが標準挙動に。あわせて `Notification` フックに BG エージェント通知（`agent_needs_input` / `agent_completed`）を追加。
- **対象**: 全プラン
- **初出**: v2.1.198 (2026-07-01)
- **情報源**: [日次レポート 7/2](../reports/2026-07-02_v2.1.198-chrome-ga-and-fable5-restored.md)

### Claude が Microsoft Foundry で GA ✅ GA
- **概要**: Claude モデルが Microsoft Foundry（Azure ホスト）で GA。Azure ネイティブの認証・課金・ガバナンス、US データゾーン対応、Opus 4.8 / Haiku 4.5 を Messages API で提供、NVIDIA GB300 Blackwell Ultra 上で稼働。CLI は `CLAUDE_CODE_USE_FOUNDRY` で指定可能。
- **対象**: エンタープライズ（Azure 利用組織）
- **初出**: 2026-06-29（Anthropic / Azure ブログ）
- **情報源**: [日次レポート 6/30](../reports/2026-06-30_v2.1.196-org-default-models-stream-watchdog-and-foundry-ga.md) / [Claude in Microsoft Foundry](https://claude.com/blog/claude-in-microsoft-foundry)

### 組織デフォルトモデル（Organization Default Models）✅ GA
- **概要**: 管理者が組織コンソールでデフォルトモデルを設定でき、ユーザー未選択時に `/model` 上で「Org default / Role default」として表示。6/13 の「利用可能モデルの強制」、6/24 の「組織モデル制限」に続く組織ガバナンス拡張。
- **対象**: Team / Enterprise
- **初出**: v2.1.196 (2026-06-29)
- **情報源**: [日次レポート 6/30](../reports/2026-06-30_v2.1.196-org-default-models-stream-watchdog-and-foundry-ga.md)

### `/clear` 実行前の地点からの `/rewind` 対応 ✅ GA
- **概要**: これまで `/clear`（履歴クリア）後は前の文脈へ戻れなかったが、`/rewind` が `/clear` 前のチェックポイントへの巻き戻し・会話再開をサポート。誤クリアの救済に。
- **対象**: 全プラン
- **初出**: v2.1.191 (2026-06-24)
- **情報源**: [日次レポート 6/25](../reports/2026-06-25_v2.1.190-191-rewind-before-clear-and-claude-tag-slack.md)

### 新スキル `/dataviz` ✅ GA
- **概要**: チャート・ダッシュボード設計のガイダンススキル。実行可能なカラーパレット検証器を同梱し配色の妥当性（コントラスト等）を検証。
- **対象**: 全プラン
- **初出**: v2.1.198 (2026-07-01)

### Claude Managed Agents のセルフホスト Sandbox ＋プライベート MCP 🧪 公開ベータ

### Claude Tag for Slack 🧪 ベータ（隣接プロダクト）
- **概要**: Slack で `@Claude` をタグ付けしタスクを委任できるチーム共有型 AI ティームメイト。段階分解して実行→スレッドへ結果投稿、チームの作業文脈を記憶、ambient モードで能動提示。既存「Claude in Slack」を置換（管理者は 30 日以内にオプトイン移行）。

### Trusted Devices for Remote Control 🧪 ベータ
- **概要**: 管理者が、メンバーのローカル Claude Code セッションをリモート閲覧・操作する前に**デバイス検証を必須化**できる組織横断設定。各ブラウザ/スマホ/デスクトップが個別に資格情報を登録（フルサインイン直後のみ提示）。デフォルトオフ。5/6 GA の Remote Agents のセキュリティ強化。
- **対象**: Team / Enterprise
- **初出**: 2026-06-25（Help Center）
- **情報源**: [日次レポート 6/28](../reports/2026-06-28_mythos5-export-clearance-and-trusted-devices.md)

## 💼 ビジネス系

### Claude Fable 5 / Mythos 5 のアクセス復旧（7/1）
- **カテゴリ**: 規制解除
- **概要**: 米商務省が 6/30 夜に輸出管理規制を解除。**公開版 Fable 5 は 7/1 に Claude Platform / Claude.ai / Claude Code / Cowork でグローバル復旧**（7/7 まで週次上限の最大 50%→以降 usage credits）。**Mythos 5 は 6/26 承認の米国機関限定で復旧**。6/10 の Fable 5 jailbreak を発端に 6/12 発令された全面停止が解除。新安全分類器が問題の jailbreak を 99% 超でブロック（CAISI 独立テスト）。
- **発生日**: 2026-06-30 夜（解除）／2026-07-01（復旧）
- **影響**: `/model fable`・`claude --model claude-fable-5` が再利用可能（要 v2.1.170+）。ただしデフォルトは Sonnet 5 のため通常ワークフローへの影響は限定的。
- **情報源**: [日次レポート 7/2](../reports/2026-07-02_v2.1.198-chrome-ga-and-fable5-restored.md) / [CoinDesk（7/1）](https://www.coindesk.com/tech/2026/07/01/anthropic-restores-ai-models-fable-mythos-after-the-u-s-lifts-export-controls)

### Mythos 5 の輸出規制部分解除（6/26）
- **カテゴリ**: 規制
- **概要**: 商務長官 Howard Lutnick の書簡で、Mythos 5 を 100+ の「信頼できる」米国機関（Project Glasswing 約 100 社＋連邦政府機関・クリティカルインフラ）へ展開する許可が下りた。書簡は公開版 Fable 5 に言及せず（当時は停止継続）。7/1 の全面復旧に先行する政策転換。
- **発生日**: 2026-06-26
- **影響**: 限定提供ラインの輸出規制ブロックを解除。7/1 の全面復旧への布石。
- **情報源**: [日次レポート 6/28](../reports/2026-06-28_mythos5-export-clearance-and-trusted-devices.md) / [CNBC（6/26）](https://www.cnbc.com/2026/06/26/us-government-anthropic-claude-mythos5-ai.html)

### Anthropic が Alibaba を蒸留目的の不正アクセスで告発（6/24）
- **カテゴリ**: 訴訟 / コンプライアンス
- **概要**: Bloomberg 報道。Anthropic は Alibaba が約 25,000 の不正アカウントで 4/22〜6/5 に 2,880 万件超のやり取りを行い、Claude の**ソフトウェアエンジニアリング／エージェント推論能力**（＝Claude Code 中核能力）をモデル蒸留目的で「不正に」アクセスしたと主張。
- **発生日**: 2026-06-24
- **影響**: フロンティアモデル能力流出を巡るガバナンス・地政学的緊張の継続。Claude Code の機能変更ではないが能力保護の背景文脈。
- **情報源**: [日次レポート 6/27](../reports/2026-06-27_v2.1.195-disable-mouse-clicks-and-alibaba-distillation.md) / [Bloomberg（6/24）](https://www.bloomberg.com/news/articles/2026-06-24/anthropic-accuses-alibaba-of-illicitly-accessing-its-ai-models)

### 6/15 サブスク課金構造変更は「施行されず一時停止」だった（訂正）
- **カテゴリ**: 価格 / 商用条件
- **概要**: 5/13-14 発表・6/15 発効予定だった programmatic 利用（Agent SDK / `claude -p` / GitHub Actions / サードパーティ Agent アプリ）の別建てドル建て monthly credit メーターへの分離計画は、**発効当日の 6/15 に Anthropic が「もはや実施しない」と公表し一時停止＝施行されなかった**ことが判明。これらの利用は引き続き通常サブスク上限から消費（変更なし）。将来変更時は事前告知するとされる。
- **発生日**: 2026-06-15（判明は 6/29 レポート）
- **影響**: 本リポジトリが「6/15 施行済み」と記録していた内容を訂正。programmatic 利用の課金は現状維持。
- **情報源**: [日次レポート 6/29](../reports/2026-06-29_june15-billing-change-paused.md) / [The New Stack](https://thenewstack.io/anthropic-pauses-claude-agent-sdk-subscription-change/)

### Model Entitlements（Enterprise ベータ・7/1）
- **カテゴリ**: エンタープライズ・ガバナンス
- **概要**: Enterprise 管理者がユーザーの利用可能モデルと effort レベル設定を制御できる新機能。既存の `availableModels` / `enforceAvailableModels` を組織コンソールのエンタイトルメントとして統合。
- **発生日**: 2026-07-01（Help Center）
- **影響**: 組織のモデル利用ガバナンスをコンソール側で一元管理。
- **情報源**: [日次レポート 7/2](../reports/2026-07-02_v2.1.198-chrome-ga-and-fable5-restored.md)
