## エグゼクティブサマリー

- **🚨 米政府の輸出管理指令により Claude Fable 5 / Mythos 5 が 6/12 に全面アクセス停止**。6/9 にローンチしたばかりの Mythos クラス 2 モデルが、jailbreak 公開を発端とする政府指令で全ユーザー無効化。週末を越えても復旧日未定が続く今週最大のニュース。**Opus 4.8 含む他全モデルは無影響**で通常ワークフローへの直接影響は限定的。
- **6/15 に予告どおり 2 つの大型変更が施行**: ① programmatic 利用（Agent SDK / `claude -p` / GitHub Actions / サードパーティアプリ）のサブスク分離（ドル建て monthly credit メーター化）、② 旧世代 Claude 4（Sonnet 4 / Opus 4）の API リタイア。
- **CLI はガバナンス・サブエージェント基盤を継続強化**: ネストサブエージェント（最大5階層、v2.1.172）、`enforceAvailableModels`（v2.1.175）、権限ルール `Tool(param:value)` 構文（v2.1.178）、`/config key=value`（v2.1.181）と、モデル統制・エージェント制御の系譜が一週間を通じて積み上がった。
- **社会・コーポレート施策が活発化**: Claude Corps（$150M 全米フェローシップ）、Economic Futures Research Fund（$200M）、Anthropic Public Record（5.2万人調査）、消費者向け本人確認ポリシー（7/8発効）。AI の雇用代替を認めた上での社会対応という一貫した文脈。
- **プラットフォーム障害が散発継続**: 6/16 で「12 日間に 10 回目」の重大障害。急成長（年商 $9B→$30B 超）に伴うインフラ逼迫が背景。

期間中の CLI リリースは 10 バージョン（うち欠番 2）。大型機能リリース（v2.1.172/175/178/181）とクリーンアップ／安定化リリース（v2.1.173/177/179）が交互に並ぶ、堅実な改善ペース。トーンとしては「機能拡張」よりも「ガバナンス強化・安定化・規制対応」が前面に出た一週間。

## 🚀 最新機能系

### ネストサブエージェント（最大 5 階層） ✅ GA
- **概要**: サブエージェントが自身のサブエージェントを生成可能に。従来の「サブエージェントは生成不可（無限ネスト防止）」制約を最大 5 階層までネイティブ緩和。Dynamic Workflows の数十〜数百並列オーケストレーションと組み合わせ深い階層分解が可能。
- **対象**: 全プラン / CLI
- **初出**: v2.1.172（2026-06-10）
- **情報源**: [日次レポート](../reports/2026-06-11_v2.1.172-nested-subagents-and-managed-agents-scheduling.md)

### 権限ルールのツールパラメータマッチ構文 `Tool(param:value)` ✅ GA
- **概要**: 権限ルールでツールの入力パラメータにマッチ可能に（`*` ワイルドカード対応）。例: `Agent(model:opus)` で Opus サブエージェントの生成をブロック。ツール名だけでなく引数値ベースのきめ細かい allow/deny を実現。
- **対象**: 全プラン / 管理者ガバナンス
- **初出**: v2.1.178（2026-06-15）
- **情報源**: [日次レポート](../reports/2026-06-16_v2.1.178-tool-param-permissions-and-june15-billing-claude4-retirement.md)

### `enforceAvailableModels` 管理設定 ✅ GA
- **概要**: `availableModels` allowlist を Default モデルにも適用し、allowlist 外解決時は先頭モデルへフォールバック。ユーザー／プロジェクト設定からの拡張（widen）も禁止する fail-closed なモデルガバナンス。Opus 4.8 デフォルト化・Fable 5 追加でモデル選択肢が増えた中での組織統制ニーズに対応。
- **対象**: エンタープライズ / Team 管理者（managed settings）
- **初出**: v2.1.175（2026-06-12）
- **情報源**: [日次レポート](../reports/2026-06-13_v2.1.174-175-enforce-available-models-and-public-record.md)

### ネスト `.claude/` ディレクトリ対応（skills 自動ロード＋最近接優先） ✅ GA
- **概要**: サブディレクトリ配下の `.claude/skills` を、そのディレクトリ配下のファイルを扱うときに自動ロード（名前衝突は `<dir>:<name>` で解決）。agent / workflow / output-style は作業ディレクトリに最も近い `.claude/` のものが優先。モノレポでのスコープ分割が容易に。
- **対象**: 全プラン / CLI
- **初出**: v2.1.178（2026-06-15）
- **情報源**: [日次レポート](../reports/2026-06-16_v2.1.178-tool-param-permissions-and-june15-billing-claude4-retirement.md)

### `/config key=value` 構文（プロンプトからの設定変更） ✅ GA
- **概要**: プロンプトから任意の設定値を直接変更できる構文（例 `/config thinking=false`）。interactive・`-p`（print）・Remote Control の全モードで動作。従来の設定画面 UI に対し、コマンド一発での設定変更を可能にした。
- **対象**: 全プラン / CLI
- **初出**: v2.1.181（2026-06-17）
- **情報源**: [日次レポート](../reports/2026-06-18_v2.1.181-config-keyvalue-and-fable5-still-suspended.md)

### Claude Managed Agents — スケジュール実行＋Vault 環境変数クレデンシャル 🧪 公開ベータ
- **概要**: ① cron スケジュールでエージェントを自動起動（スケジューラ自前構築不要、一時停止／再開／手動トリガー可）、② シークレットをサンドボックスにはプレースホルダのみ配置しネットワーク境界でのみ実鍵を注入（プロンプトインジェクション耐性）。**追加課金なし**。「人間がいないとき誰がエージェントを起動するか」というエンタープライズ導入障壁を解消。
- **対象**: Claude Platform / Claude Platform on AWS
- **初出**: 2026-06-09 発表（6/11 レポートで反映）
- **情報源**: [日次レポート](../reports/2026-06-11_v2.1.172-nested-subagents-and-managed-agents-scheduling.md)

### 会話言語ベースのセッションタイトル自動生成 ✅ GA
- **概要**: セッションタイトルが会話で使われている言語で生成されるように（`language` 設定でピン留め可）。日本語など非英語ユーザーのセッション一覧視認性を改善。
- **対象**: 全プラン / CLI
- **初出**: v2.1.176（2026-06-12）
- **情報源**: [日次レポート](../reports/2026-06-14_v2.1.176-177-and-fable5-mythos5-government-suspension.md)

## 💼 ビジネス系

### 🚨 米政府の輸出管理指令により Claude Fable 5 / Mythos 5 が全面アクセス停止
- **カテゴリ**: 規制 / 障害（アクセス停止）
- **概要**: 6/9 ローンチの Claude Fable 5（✅ GA）と Mythos 5（🔬 限定提供）が、6/12 17:21 ET の米政府輸出管理指令（米国内外問わず外国籍者によるアクセス停止を要求）への即時コンプライアンスのため全ユーザーで無効化。発端は 6/10 に X で公開された Fable 5 jailbreak。Anthropic は「誤解だと考えており、できる限り早く復旧する」と不同意を表明。期間末（6/18）時点も**復旧日未定**。
- **発生日**: 2026-06-12（6/16・6/17・6/18 時点も継続）
- **影響**: Mythos クラスは事実上アクセス不可へ逆戻り。**ただし Opus 4.8・Sonnet 4.6・Haiku 4.5 など他全モデルは無影響**で、`/model fable` がエラー／フォールバックになる以外、通常のコーディングワークフローへの直接影響は限定的。US 市民は 7/8 発効の本人確認経由で US 限定再開し得るとの観測。
- **情報源**: [6/14 日次](../reports/2026-06-14_v2.1.176-177-and-fable5-mythos5-government-suspension.md) / [6/16 日次](../reports/2026-06-16_v2.1.178-tool-param-permissions-and-june15-billing-claude4-retirement.md) / [6/18 日次](../reports/2026-06-18_v2.1.181-config-keyvalue-and-fable5-still-suspended.md)

### 6/15 サブスクリプション課金構造変更が施行
- **カテゴリ**: 価格 / 商用条件
- **概要**: programmatic な Claude 利用（Claude Agent SDK / `claude -p` ヘッドレス / Claude Code GitHub Actions / サードパーティ Agent アプリ）がサブスクの flat-rate プールから完全分離され、**ユーザー単位のドル建て monthly credit メーター**（標準 API レート課金・ロールオーバーなし・チーム間プールなし）へ移行。目安は Pro 約 $20・Max 5x 約 $100・Max 20x 約 $200/月。枯渇後は overflow billing を手動有効化しない限り自動リクエスト停止。
- **発生日**: 2026-06-15 施行（5/14 既報 → 施行済み）
- **影響**: ターミナルでの対話的 Claude Code 利用・claude.ai web/desktop/mobile チャット・Cowork はサブスク内で継続。自動化／CI ワークフローを多用する利用者はコスト構造の見直しが必要。
- **情報源**: [6/16 日次](../reports/2026-06-16_v2.1.178-tool-param-permissions-and-june15-billing-claude4-retirement.md) / [6/13 日次](../reports/2026-06-13_v2.1.174-175-enforce-available-models-and-public-record.md)

### 旧世代 Claude 4（Sonnet 4 / Opus 4）API リタイアが施行
- **カテゴリ**: 商用条件 / モデルライフサイクル
- **概要**: 米国太平洋時間 6/15 9:00 AM に `claude-sonnet-4-20250514` / `claude-opus-4-20250514` を API からリタイア。以後これらのモデル ID への全リクエストはエラー応答（猶予期間・silent fallback なし）。移行先は Sonnet 4.6 / Opus 4.8。研究者は External Researcher Access Program で継続申請可。
- **発生日**: 2026-06-15 施行（4/14 廃止予告済み）
- **影響**: コードベース内の `20250514` を全文検索して露出箇所を更新するのが最短の対応策。
- **情報源**: [6/16 日次](../reports/2026-06-16_v2.1.178-tool-param-permissions-and-june15-billing-claude4-retirement.md) / [6/14 日次](../reports/2026-06-14_v2.1.176-177-and-fable5-mythos5-government-suspension.md)

### Claude Corps（$150M 全米フェローシップ）
- **カテゴリ**: 社会施策 / 人材
- **概要**: $150M を投じる全米フェローシップ。1,000 人を 3 コホートで養成し非営利団体に 1 年常駐させ、年俸 $85,000 + 福利厚生を支給。第 1 コホート 100 人は 10 月開始、応募締切 7/17、ホスト非営利は初年度 400+ 団体。
- **発生日**: 2026-06-11（✅ 開始・応募受付中）
- **影響**: Anthropic が「自社 AI が労働を代替しうる」ことを認めた上での社会的施策という文脈。
- **情報源**: [6/12 日次](../reports/2026-06-12_v2.1.173-claude-corps-and-economic-futures-fund.md)

### Economic Futures Research Fund（$200M AI 経済影響研究基金）
- **カテゴリ**: 政策 / 研究投資
- **概要**: $200M を AI の経済的影響に関する研究トライアル・公共政策評価へ拠出（ガバナンス・配分詳細は未公表）。同時に Dario Amodei が AI 起因失業の段階的政策フレームワーク（5% / 10% / 前例なきレベル → UBI・ソブリンウェルス・エクイティ共有）を提言。
- **発生日**: 2026-06-10（6/12 レポートで反映、📢 発表のみ）
- **影響**: 後述の Anthropic Public Record・Claude Corps と整合する社会的文脈づくり。Gates Foundation $200M 提携（5/14）とは別件。
- **情報源**: [6/12 日次](../reports/2026-06-12_v2.1.173-claude-corps-and-economic-futures-fund.md)

### 消費者向けプランに本人確認・年齢確認ポリシー導入（7/8 発効）
- **カテゴリ**: コンプライアンス / プライバシー
- **概要**: Claude Free / Pro / Max にプライバシーポリシー改定で年齢・本人確認を求める権利を明文化。年齢確認は Yoti（セルフィー推定／身分証／18+ クレデンシャル）、本人確認は Persona（政府発行写真 ID + ライブセルフィー）。ID 画像は Persona/Yoti 側保持・自社サーバ非保存・暗号化・広告／学習に不使用。Team / Enterprise / Platform は対象外。
- **発生日**: 2026-06-15 公表 / 2026-07-08 発効（📢 発表のみ）
- **影響**: Claude Code をサブスク認証で使う消費者プラン利用者には間接的影響の可能性。Fable 5 の US 限定再開枠組みになり得るとの観測も。
- **情報源**: [6/17 日次](../reports/2026-06-17_v2.1.179-and-id-verification-policy.md)

### Anthropic Public Record（5.2万人 AI 意識調査）
- **カテゴリ**: 政策 / 調査
- **概要**: 約 52,000 人の米国民を対象にした AI への期待・不安の初回大規模調査結果を公開。期待 1 位は「疾病治療」48%、一方で全州で最多の不安は「AI による失業」64%。
- **発生日**: 2026-06-12（📢 公式）
- **影響**: Economic Futures Research Fund・Claude Corps と一体の社会的文脈形成。
- **情報源**: [6/13 日次](../reports/2026-06-13_v2.1.174-175-enforce-available-models-and-public-record.md)

### Claude プラットフォーム障害の継続
- **カテゴリ**: 障害 / インフラ
- **概要**: 6/16 で 6/5 以降「12 日間に 10 回目」の重大障害（Opus 4.8 / Haiku 4.5 のエラー率上昇）。Anthropic は前例なき需要によるインフラ逼迫を認める（年商 $9B→$30B 超、$1M 超エンタープライズ顧客が 2 ヶ月で 500→1,000 社）。6/17 時点では直近インシデントは解消済み・全体は operational。
- **発生日**: 2026-06-05 以降継続（6/16 が直近）
- **影響**: 急成長フェーズにおける信頼性リスクが顕在化。claude.ai / API / Claude Code / Cowork 全体に影響。
- **情報源**: [6/17 日次](../reports/2026-06-17_v2.1.179-and-id-verification-policy.md) / [6/18 日次](../reports/2026-06-18_v2.1.181-config-keyvalue-and-fable5-still-suspended.md)
