## エグゼクティブサマリー

- **🏆 Mythos クラスモデルがついに一般提供（GA）開始** — 6/9、長らく「噂 → 発表のみ」と追跡してきた Mythos クラス general release が **「Claude Fable 5」** として確定（📢 → ✅）。SWE-bench Verified 95.0% の最強公開モデルで、Claude Code から `/model fable` で選択可能（要 v2.1.170+、デフォルトは Opus 4.8 据え置き）。
- **🤖 エージェント機能が一段深化** — v2.1.172 で **ネストサブエージェント（最大 5 階層）** がネイティブ解禁。Managed Agents には **スケジュール実行（cron 自動起動）** と **Vault クレデンシャル** が公開ベータで追加され、「無人運用」の障壁を解消。
- **🛡 安全解除版 Claude Mythos 5（🔬 限定）** が Project Glasswing パートナー・生物学研究者向けに同時投入。Glasswing は約 150 組織・15+ か国へ拡大（NATO / ENISA / Samsung 等）。
- **💰 資金・上場が大きく前進** — 5/28 に **Series H $65B 調達 / $965B post-money** がクローズ（従来の $30B/$900B 噂を上回る確定額）、6/1 に **SEC へ S-1 を機密申請（IPO 準備）**。
- **🔧 信頼性・トラブルシュート強化** — `--safe-mode` / `/cd` / `fallbackModel`（最大 3 段）等の運用支援機能と、1M コンテキスト枯渇時の恒久スタック解消など 30+ のバグ修正。

期間全体のトーンは「**Mythos クラスの民主化 + エージェント自律運用の本格化**」。新モデル投入・上場準備・クリティカルインフラ展開が同時並行で進み、製品とビジネスの両面で勢いが極めて強い 1 週間だった。

## 🚀 最新機能系

### Claude Fable 5 — Mythos クラスの一般提供モデル ✅ GA
- **概要**: Opus の上位「Mythos クラス」の安全版を一般公開。SWE-bench Verified **95.0%** / SWE-bench Pro **80.3%**（GPT-5.5 58.6%）/ Terminal-Bench 2.1 **88.0%** / FrontierCode Diamond **29.3%**。機微リクエストは Opus 4.8 へ自動フォールバック（発動 5% 未満）。Stripe は 5,000 万行 Ruby 移行（約 2 ヶ月見積もり）を **1 日**に短縮。
- **対象**: Pro / Max / Team / Enterprise（6/22 まで追加課金なし、6/23 以降は容量次第で usage credits）/ API / Amazon Bedrock（launch 同時 GA、N. Virginia / Stockholm）。価格 $10/$50 per MTok、1M コンテキスト、128k 出力。
- **Claude Code 連携**: `/model fable`（または `--model claude-fable-5`）で選択。**要 CLI v2.1.170+**。デフォルトモデルは Opus 4.8 のままで、Fable 5 は難タスク向け「エスカレーション用」。
- **初出**: v2.1.170（2026-06-09）
- **情報源**: [日次レポート 06-10](../reports/2026-06-10_claude-fable-5-mythos-5-ga-and-v2.1.170.md) / [Anthropic - Claude Fable 5 and Mythos 5](https://www.anthropic.com/news/claude-fable-5-mythos-5)

### Claude Opus 4.8 — Claude Code のデフォルトモデルに ✅ GA
- **概要**: 5/28 に GA。SWE-bench Verified 88.6% / SWE-bench Pro 69.2%（GPT-5.5・Gemini 3.1 Pro 超え）。標準価格据え置き（$5/$25）、fast mode は $10/$50 で従来比 **3 倍安価・2.5x 高速**。コード欠陥の見落としが前世代比約 1/4 の「near-Mythos alignment」。
- **対象**: Max / Team Premium / Enterprise / API でデフォルト（effort は high）。CLI / Desktop / VS Code / Bedrock / Vertex / Foundry。
- **初出**: v2.1.154（2026-05-28、デフォルト化）
- **情報源**: [日次レポート 06-08](../reports/2026-06-08_opus-4.8-default-dynamic-workflows-and-v2.1.151-168.md) / [Anthropic - Introducing Claude Opus 4.8](https://www.anthropic.com/news/claude-opus-4-8)

### ネストサブエージェント（最大 5 階層）✅ GA
- **概要**: サブエージェントが自身のサブエージェントを生成可能に。従来の「無限ネスト防止のため生成不可」という制約を **最大 5 階層までネイティブ緩和**。Dynamic Workflows の数十〜数百並列オーケストレーションと組み合わせ、深い階層分解が可能に。
- **対象**: 全プラン（CLI 安定版）
- **初出**: v2.1.172（2026-06-10）
- **情報源**: [日次レポート 06-11](../reports/2026-06-11_v2.1.172-nested-subagents-and-managed-agents-scheduling.md)

### Dynamic Workflows（動的ワークフロー）🔬 研究プレビュー
- **概要**: Claude が単一セッション内でタスクを分解し、**数十〜数百の並列サブエージェント**にオーケストレーション・検証してから最終回答を提示。コードベース移行など大規模作業向け。effort メニューの `ultracode` でアクセス。
- **対象**: Enterprise / Team / Max（Opus 4.8 環境）/ CLI・Desktop・VS Code・API・Bedrock・Vertex・Foundry。
- **初出**: v2.1.154（2026-05-28）
- **情報源**: [日次レポート 06-08](../reports/2026-06-08_opus-4.8-default-dynamic-workflows-and-v2.1.151-168.md) / [InfoQ - Dynamic Workflows](https://www.infoq.com/news/2026/06/dynamic-workflows-claude-code/)

### Claude Managed Agents — スケジュール実行（Scheduled Deployments）🧪 公開ベータ
- **概要**: **cron スケジュールでエージェントを自動起動**。スケジューラの自前構築・ホスト不要で、一時停止・再開・手動トリガーが可能。「人間がいないとき誰がエージェントを起動するのか」というエンタープライズ導入の障壁を解消。**追加課金なし**。早期採用: Rakuten / Notion / Browserbase 他。
- **対象**: Claude Platform / Claude Platform on AWS
- **初出**: 2026-06-09（プラットフォーム発表）
- **情報源**: [日次レポート 06-11](../reports/2026-06-11_v2.1.172-nested-subagents-and-managed-agents-scheduling.md) / [Anthropic - Scaling Managed Agents](https://www.anthropic.com/engineering/managed-agents)

### Claude Managed Agents — Vault 環境変数クレデンシャル 🧪 公開ベータ
- **概要**: シークレットを Vault に保管し、サンドボックスには**プレースホルダのみ**配置。実鍵は**ネットワーク境界（allowlist ドメイン宛リクエスト）でのみ注入**。プロンプトインジェクションでもモデルの context に実シークレットが載らない。**追加課金なし**。
- **対象**: Claude Platform / Claude Platform on AWS
- **初出**: 2026-06-09（プラットフォーム発表）
- **情報源**: [日次レポート 06-11](../reports/2026-06-11_v2.1.172-nested-subagents-and-managed-agents-scheduling.md) / [Managed Agents vaults docs](https://platform.claude.com/docs/en/managed-agents/vaults)

### Claude Mythos 5 — 安全分類器を解除した限定提供版 🔬 限定提供
- **概要**: Fable 5 と同一基盤でサイバーセキュリティ分野の安全分類器を解除。創薬を 10 倍加速、AAV 設計で専用タンパク質モデルを凌駕。デュアルユース性ゆえアクセス厳格制限。Bedrock は Limited Preview。
- **対象**: Project Glasswing パートナー + 選定された生物学研究者（trusted access）
- **初出**: 2026-06-09
- **情報源**: [日次レポート 06-10](../reports/2026-06-10_claude-fable-5-mythos-5-ga-and-v2.1.170.md) / [TechCrunch](https://techcrunch.com/2026/06/09/anthropic-released-claude-fable-5-its-most-powerful-model-publicly-days-after-warning-ai-is-getting-too-dangerous/)

### security-guidance プラグイン ✅ GA / 全プラン無料
- **概要**: コード編集・モデル出力・コミットをリアルタイムに自律レビューする 3 層防御（毎編集のパターンマッチ 25 種 → git diff のモデルレビュー → commit/push 時のエージェント型レビュー）。内部テストでセキュリティ関連 PR コメントを 30〜40% 削減。`/plugin install security-guidance@claude-plugins-official`。
- **対象**: 全プラン（要 CLI 2.1.144+ / Python 3.8+）
- **初出**: 2026-05-26（期間内レポートで初整理）
- **情報源**: [日次レポート 06-08](../reports/2026-06-08_opus-4.8-default-dynamic-workflows-and-v2.1.151-168.md) / [CybersecurityNews](https://cybersecuritynews.com/free-security-plugin-for-claude-code/)

### トラブルシュート／信頼性向上フラグ群 ✅ GA
- **概要**: `--safe-mode`（全カスタマイズ無効で起動）、`/cd`（プロンプトキャッシュを壊さず作業ディレクトリ変更）、`disableBundledSkills`（同梱スキル・コマンド非表示）、`fallbackModel`（最大 3 段フォールバック）、thinking 無効化（`MAX_THINKING_TOKENS=0` / `--thinking disabled`）。プラグイン拡張（`.claude/skills` 自動ロード、`claude plugin init`、`/plugin` マーケットプレイス検索バー）。
- **対象**: 全プラン
- **初出**: v2.1.166（fallbackModel 等）/ v2.1.169（safe-mode・cd）/ v2.1.172（検索バー）
- **情報源**: [日次レポート 06-08](../reports/2026-06-08_opus-4.8-default-dynamic-workflows-and-v2.1.151-168.md) / [日次レポート 06-09](../reports/2026-06-09_v2.1.169-safe-mode-cd-and-glasswing-150-orgs.md)
