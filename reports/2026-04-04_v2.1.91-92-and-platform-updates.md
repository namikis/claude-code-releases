# Claude Code v2.1.91〜v2.1.92 & プラットフォーム更新

**調査日:** 2026-04-04
**対象期間:** 2026-04-02〜2026-04-04
**対象バージョン:** v2.1.91, v2.1.92

---

## 概要

前回調査（2026-04-02、v2.1.90まで）以降、2つの新バージョンがリリースされた。v2.1.91ではMCPツール結果の永続化上限引き上げやスキルのシェル実行無効化設定が追加され、v2.1.92ではエンタープライズ向けの`forceRemoteSettingsRefresh`ポリシー設定、AWSインタラクティブセットアップウィザード、`/cost`コマンドの詳細化、`/release-notes`コマンドの改善などが加わった。API/プラットフォーム側に4月の新規リリースノートはないが、Claude Haiku 3の廃止日（4/19）が間近に迫っている。公式ブログには「Harnessing Claude's Intelligence」（4/2）が投稿されたが、機能発表ではなくエージェント設計パターンの解説記事。

---

## 主要な新機能・変更

### v2.1.92（2026-04-04）

#### `forceRemoteSettingsRefresh` ポリシー設定
- **リリース状態**: GA
- 起動時にリモートmanaged settingsを必ず最新取得し、取得失敗時はエラー終了（fail-closed）
- エンタープライズ環境でのポリシー適用の確実性を強化

#### Bedrock インタラクティブセットアップウィザード
- **リリース状態**: GA
- ログイン画面からAWS認証、リージョン設定、認証情報検証、モデルピンニングを対話的に設定
- AWS Bedrock経由でClaude Codeを利用するユーザーのオンボーディングを大幅に簡素化

#### `/cost` コマンド強化
- **リリース状態**: GA（サブスクリプションユーザー向け）
- モデル別・キャッシュヒット別のコスト内訳を表示
- 使用状況の可視性向上

#### `/release-notes` インタラクティブバージョンピッカー
- **リリース状態**: GA
- バージョンを選択してリリースノートを閲覧するインタラクティブUIに変更

#### Remote Control セッション名のデフォルトプレフィックス
- ホスト名をデフォルトプレフィックスに使用（例: `myhost-graceful-unicorn`）
- `--remote-control-session-name-prefix` で上書き可能

#### プロンプトキャッシュ期限切れヒント（Proユーザー）
- セッションに戻った際、プロンプトキャッシュが期限切れの場合にフッターでヒント表示

#### `/tag` コマンド廃止
- 削除された

#### `/vim` コマンド廃止
- 削除。エディターモードの切替は `/config` → Editor mode から行う

#### Linux sandbox `apply-seccomp` ヘルパー
- npm / ネイティブビルドに `apply-seccomp` ヘルパーを同梱

#### バグ修正
- tmuxウィンドウがkill/renumber後にサブエージェントスポーンが「Could not determine pane count」で失敗する問題を修正
- `prompt-type` Stopフックが小型fastモデルで`ok:false`を返した場合に失敗する問題を修正
- ストリーミングが配列/オブジェクトフィールドをJSON文字列として出力した際のツール入力バリデーションエラーを修正
- Extended Thinkingが空白のみのテキストブロックを生成した際のAPI 400エラーを修正
- auto-pilotキー入力によるフィードバックサーベイの誤送信を修正
- フルスクリーンモードの「esc to interrupt」ヒント表示の誤りを修正
- Homebrew caskチャンネルの更新プロンプトを修正
- `ctrl+e` がマルチラインプロンプトの行末で次の行にジャンプする問題を修正
- iTerm2、Ghostty、DEC 2026対応ターミナルでのスクロールバック位置ズレを修正
- アイドル復帰ヒントがセッション累計トークン数を表示していた問題を修正（現在のコンテキストサイズに変更）
- プラグインMCPサーバーが未認証claude.aiコネクタ重複時に「connecting」で停止する問題を修正

#### パフォーマンス改善
- Writeツールのdiff計算速度を60%向上（タブ・`&`・`$`を含む大きなファイルで効果大）

### v2.1.91（2026-04-02）

#### MCP ツール結果永続化オーバーライド
- **リリース状態**: GA
- `_meta["anthropic/maxResultSizeChars"]` アノテーションで最大500Kまでの大きな結果（DBスキーマ等）を截断せず保持可能に

#### `disableSkillShellExecution` 設定
- **リリース状態**: GA
- スキル、カスタムスラッシュコマンド、プラグインコマンド内のインラインシェル実行を無効化
- セキュリティポリシーが厳しい環境向け

#### ディープリンクのマルチライン対応
- `claude-cli://open?q=` ディープリンクでエンコードされた改行（`%0A`）が拒否されなくなった

#### プラグイン実行可能ファイル
- プラグインが `bin/` 配下に実行ファイルを同梱し、Bashツールからベアコマンドとして呼び出し可能に

#### バグ修正
- `--resume` でのトランスクリプトチェーン断裂（非同期書き込み失敗時に会話履歴が失われる）を修正
- `cmd+delete` がiTerm2、kitty、WezTerm、Ghostty、Windows Terminalで行頭まで削除できない問題を修正
- リモートセッションのPlan Modeでコンテナ再起動後にプランファイルの追跡が失われる問題を修正
- `permissions.defaultMode: "auto"` のJSON Schema バリデーションを修正
- Windowsバージョンクリーンアップがアクティブバージョンのロールバックコピーを保護しない問題を修正

#### 改善
- `/feedback` が利用不可時にスラッシュメニューから消えるのではなく理由を表示するように変更
- `/claude-api` スキルのエージェント設計パターンガイダンスを改善
- Bunでの`stripAnsi` パフォーマンス向上（`Bun.stripANSI`経由）
- Editツールが短い`old_string`アンカーを使用し、出力トークンを削減
