# 調査レポート: Claude Code v2.1.176 / v2.1.177 + 米政府指令による Claude Fable 5 / Mythos 5 全面アクセス停止

**調査日:** 2026-06-14
**調査モード:** フルモード（Layer 1〜3）

## 概要

6/13 レポート（v2.1.174 / v2.1.175）以降の差分を Layer 1〜3 で調査。

**Layer 1** では CLI **v2.1.176（6/12 21:53 UTC）** と **v2.1.177（6/13 01:25 UTC、CHANGELOG/feed.xml の chore のみ）** の 2 リリースを確認。v2.1.176 が実質的な更新で、目玉は **会話言語に合わせたセッションタイトル自動生成（`language` 設定でピン留め可）** と **`footerLinksRegexes` 設定（正規表現マッチでフッター行にリンクバッジ表示）**、および **Bedrock クレデンシャルキャッシュ改善**（`awsCredentialExport` の認証情報を固定 1 時間ではなく `Expiration` までキャッシュ）。加えて `availableModels` の allowlist 強制を `ANTHROPIC_DEFAULT_*_MODEL` 環境変数経由のすり抜けに対しても閉じる修正、Hook の `if` 条件（Read/Edit/Write のパスマッチ）修正、Linux サンドボックスの symlink 設定対応、tmux over SSH のクリップボード修正など多数のバグ修正。**新スラッシュコマンド・新 Hook イベントの追加はなし**。**v2.1.171 は引き続き欠番**、Layer 1 最新は **v2.1.177**。

**Layer 2/3** では極めて大きなニュース ──  **米国政府の輸出管理指令により、6/9 にローンチしたばかりの Claude Fable 5 と Claude Mythos 5 が 6/12 に全ユーザー向けにアクセス停止**された。指令は「米国内外を問わずあらゆる外国籍者によるアクセスの停止」を要求し、Anthropic は外国籍ユーザーをリアルタイムに選別できないため**全ユーザーで両モデルを無効化**。発端は 6/10 に X 上で公開された Fable 5 の jailbreak（サイバー攻撃・爆発物・化学合成経路の実用情報を引き出したとの主張）。**Opus 4.8 を含む他の全モデルは影響なし**。これにより前回まで ✅ GA / 🔬 限定提供だった Mythos クラスモデルは**事実上アクセス不可に逆戻り**した。あわせて **6/15 のサブスク課金構造変更が翌日に迫り**、同日に**旧世代 Claude 4 モデル 2 種の API リタイア**も確定。

## 主要な新機能（v2.1.176）

### 会話言語に合わせたセッションタイトル自動生成 ✅
- **リリース状態**: ✅ GA
- **概要**: セッションタイトルが**会話で使われている言語**で生成されるようになった。`language` 設定で特定言語にピン留め可能。日本語など非英語ユーザーの一覧視認性を改善。
- **初出**: v2.1.176 (2026-06-12)
- **情報源**: [Claude Code changelog](https://code.claude.com/docs/en/changelog)

### `footerLinksRegexes` 設定 ✅
- **リリース状態**: ✅ GA（設定）
- **概要**: フッター行に**正規表現マッチでリンクバッジ**を表示する設定。ユーザー設定・管理設定の両方から構成可能。CI/PR・社内ダッシュボード等への動的リンク表示を想定。
- **初出**: v2.1.176 (2026-06-12)
- **情報源**: [Claude Code changelog](https://code.claude.com/docs/en/changelog)

### Bedrock クレデンシャルキャッシュ改善 ✅
- **リリース状態**: ✅ GA
- **概要**: `awsCredentialExport` から取得した AWS 認証情報を、従来の**固定 1 時間**ではなく**実際の `Expiration` まで**キャッシュするように改善。短命/長命いずれのクレデンシャルでも不要な再取得や早期失効を回避。
- **初出**: v2.1.176 (2026-06-12)
- **情報源**: [Claude Code changelog](https://code.claude.com/docs/en/changelog)

## バグ修正・パフォーマンス改善（v2.1.176 主要分）

- **`availableModels` 強制のすり抜け修正** — エイリアスのモデル選択が `ANTHROPIC_DEFAULT_*_MODEL` 環境変数経由でブロック済みモデルへリダイレクトされないように。v2.1.175 の `enforceAvailableModels` に続くガバナンス強化。
- **`/fast` がトグル拒否する問題修正** — allowlist 外のモデルへ切り替える際に `/fast` が反応しない問題を解消。
- **Auto mode の Fable 5 失敗修正** — Opus 4.8 を有効化していない組織で Auto mode が Fable 5 で失敗する問題を修正。分類器が利用可能な最良の Opus モデルにフォールバックするように。
- **Hook の `if` 条件（ツールパスマッチ）修正** — `Edit(src/**)` / `Read(~/.ssh/**)` / `Read(.env)` のような Read/Edit/Write パス条件が正しく評価されるように。
- **Linux サンドボックス起動失敗修正** — `.claude/settings.json` が絶対パスターゲットの symlink の場合に起動失敗する問題を修正。
- **クリップボード修正** — tmux over SSH 内で `/copy` とマウス選択コピーがシステムクリップボードに届かない問題を修正。
- **Remote Control 修正群** — 接続通知・アカウント切替・セッション切断、および web/mobile からのサイレントなセッションモデル切替を修正。
- **バックグラウンドセッション修正群** — `/cd`、ウィンドウナビゲーション、ターン途中の `/bg` で継続対象がない場合に永遠に「Working」表示になる問題、PR URL 検索、Windows ネットワークパスなど多数。
- **Cloud セッション** — アイドルタイムアウト時の認証失敗を修正。

### v2.1.177
- `chore: Update CHANGELOG.md and feed.xml` のみ（機能変更なし）。

## Layer 2/3: プラットフォーム・コーポレート動向

### 🚨 米政府の輸出管理指令により Claude Fable 5 / Mythos 5 が全面アクセス停止（6/12）

- **確度**: ✅ 公式（Anthropic 公式声明 + 複数大手メディア）
- **ステータス変更**: **Claude Fable 5 ✅ GA → ⛔ アクセス停止 / Claude Mythos 5 🔬 限定提供 → ⛔ アクセス停止**
- **経緯**:
  - **6/9**: Claude Fable 5（✅ GA）と Claude Mythos 5（🔬 限定提供）をローンチ。
  - **6/10**: ある jailbreaker が X 上で Fable 5 のセーフガードを破ったと主張する jailbreak を公開（サイバー攻撃・爆発物・化学合成経路の実用的指示を引き出したと主張）。
  - **6/12 17:21 ET**: 米国政府が**国家安全保障当局を根拠とする輸出管理指令**を発出。「**米国内外を問わずあらゆる外国籍者（外国籍の Anthropic 従業員を含む）によるアクセスの停止**」を要求。
  - Anthropic は外国籍ユーザーを米国ユーザーからリアルタイムに選別する手段がないため、**コンプライアンス確保のため全ユーザー向けに両モデルを無効化**。
- **Anthropic の立場**: 「狭い範囲の潜在的 jailbreak の発見が、数億人に展開された商用モデルのリコール理由になるべきではない」と**不同意**を表明。指令の根拠となった jailbreak は「特定のコードベースを読ませて software flaw を修正させる」程度のもので、示された能力レベルは**他社モデルでも広く利用可能**だと主張。「return access as quickly as we can（できる限り早くアクセスを復旧する）」とコメント。
- **影響範囲**: **Opus 4.8 を含むその他全 Anthropic モデルは影響なし**。Claude Code・API・Bedrock 上の Opus 4.8 / Sonnet 4.6 / Haiku 4.5 等は通常どおり利用可能。
- **Claude Code への含意**: 6/9 の v2.1.170 で追加された `/model fable`（Fable 5 選択）は当面**呼び出してもエラー/フォールバックになる**見込み。デフォルトモデルは元々 Opus 4.8 のため、通常のコーディングワークフローへの直接影響は限定的。
- **情報源**: [Anthropic 公式声明](https://www.anthropic.com/news/fable-mythos-access) / [VentureBeat](https://venturebeat.com/technology/anthropic-blocks-all-public-access-to-claude-fable-5-mythos-5-following-us-government-order-what-enterprises-should-do) / [MarkTechPost](https://www.marktechpost.com/2026/06/13/anthropic-disables-claude-fable-5-and-mythos-5-after-us-government-order/) / [gHacks](https://www.ghacks.net/2026/06/13/us-government-suspends-anthropics-claude-fable-5-and-mythos-5-over-security-and-jailbreak-concerns/)

### 6/15 サブスク課金構造変更 + Claude 4 リタイアが翌日に迫る 📢

- **確度**: ✅ 公式（5/14 既報、6/15 施行前日）
- **課金変更**: **2026-06-15 から** programmatic な Claude 利用（**Claude Agent SDK / `claude -p` CLI / Claude Code GitHub Actions / サードパーティ Agent アプリ**）がサブスクの flat-rate プールから分離され、**ドル建ての別建て monthly credit メーター（標準 API レート課金・ロールオーバーなし）**へ移行。目安は **Pro 約 $20/月・Max 5x 約 $100/月・Max 20x 約 $200/月**。クレジット枯渇時は overflow billing を有効化しない限り自動リクエストが停止。**ターミナルの対話的 Claude Code 利用・claude.ai web/desktop チャットはサブスク内で継続**。
- **Claude 4 リタイア（同日 6/15）**: 旧世代 Claude 4 モデル 2 種を API からリタイア。**猶予期間なし**で切替（要マイグレーション）:
  - `claude-sonnet-4-20250514` → `claude-sonnet-4-6`
  - `claude-opus-4-20250514` → `claude-opus-4-8`
- **情報源**: [The New Stack](https://thenewstack.io/anthropic-agent-sdk-credits/) / [Enterprise DNA](https://enterprisedna.co/resources/news/anthropic-claude-june-15-retirements-billing-2026/) / [Codersera](https://codersera.com/blog/anthropic-june-2026-billing-change-claude-code/)
