# 調査レポート: v2.1.105・Word Add-in GA・Opus 4.7リーク

**調査日**: 2026-04-14
**調査モード**: フルモード（Layer 1〜3）
**対象期間**: 2026年4月13〜14日
**新CLIリリース**: v2.1.105（2026年4月13日）

---

## 概要

v2.1.105がリリースされ、PreCompactフックのブロッキング対応、プラグインバックグラウンドモニター、EnterWorktreeの既存worktreeサポート等が追加された。ニュース面では、Claude for Word Add-inのGA化によりMicrosoft Office完全統合が達成された。また、Claude Opus 4.7のリーク情報とAnthropicの「Full-Stack AI Studio」構想が噂として報じられた。英国AI安全性研究所がMythosの自律サイバー攻撃能力を評価した結果が公開された。

---

## Layer 1: CLIリリース

### v2.1.105（2026年4月13日）

#### 新機能

| 機能 | 詳細 |
|------|------|
| **EnterWorktree `path`パラメータ** | 既存のworktreeに切り替え可能に。新規作成だけでなく、既存worktreeへのスイッチ操作をサポート |
| **PreCompactフック ブロッキング対応** | PreCompactフックがexit code 2または`{"decision":"block"}`でコンパクションをブロック可能に |
| **プラグイン バックグラウンドモニター** | プラグインmanifestにトップレベル`monitors`キーを追加。セッション開始時またはスキル呼び出し時に自動起動 |
| **`/proactive`→`/loop`エイリアス** | `/proactive`が`/loop`のエイリアスに |

#### 改善

| 改善 | 詳細 |
|------|------|
| **APIストリーム停止ハンドリング** | 5分間データなしでストリーム中断→非ストリーミングリトライ（無限ハングを解消） |
| **ネットワークエラーメッセージ** | 接続エラー時にサイレントスピナーではなく即座にリトライメッセージ表示 |
| **ファイル書き込み表示** | minified JSON等の長い単一行書き込みをUIで切り詰め表示（大量ページネーション解消） |
| **`/doctor`レイアウト** | ステータスアイコン追加。`f`キーでClaude自動修正 |
| **スキル説明文キャップ** | 一覧表示の上限を250→1,536文字に拡大。起動時に切り詰め警告を表示 |
| **WebFetch `<style>`/`<script>`除去** | CSSヘビーなページでコンテンツバジェットが実テキスト到達前に消費される問題を解消 |
| **古いworktreeクリーンアップ** | squash-mergeされたPRのworktreeも自動削除（従来は残留） |
| **MCP大出力切り詰めプロンプト** | JSON向け`jq`レシピ、テキスト向けReadチャンクサイズ等、フォーマット固有のガイダンス |

#### バグ修正（主要なもの）

- キュー済みメッセージの添付画像が破棄される問題を修正
- プロンプト入力が2行に折り返すと画面が空白になる問題を修正
- アシスタントメッセージの先頭空白切り詰め（ASCIIアート・インデント図の破損）を修正
- Python `rich`/`loguru`等のクリック可能ファイルリンクでbash出力が文字化けする問題を修正
- `alt+enter`改行挿入のリグレッション（v2.1.100〜）を修正
- 一回限りのスケジュールタスクがファイルウォッチャーのクリーンアップ見逃しで繰り返し発火する問題を修正
- Team/Enterpriseユーザーで最初のメッセージ以降のチャネル通知がサイレントドロップされる問題を修正
- マーケットプレイスプラグインの依存関係自動インストール、自動更新時の破損を修正
- stdio MCPサーバーの不正JSON出力でセッションがハングする問題を修正（フェイルファスト化）
- ヘッドレス/リモートトリガーセッションでMCPツールが最初のターンで使用不可になる問題を修正
- Bedrock非US地域で`/model`ピッカーが無効な`us.*`モデルIDを永続化する問題を修正
- 429レートリミットエラーが生JSONダンプで表示される問題を修正
- Ghostty、Kitty、Alacritty、WezTerm等SSH/mosh経由で16色パレットが薄くなる問題を修正
- `CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC`が全プロジェクトのメトリクスを永久無効化する問題を修正
- その他多数のUI・プラグイン・権限関連修正

## Layer 3: ニュースメディア

### 1. Claude for Word Add-in — Microsoft Office完全統合 ✅

**確度**: ✅ GA（公式プレスリリース）
**日付**: 2026年4月13日

AnthropicがClaude for Word Add-inをリリースし、Microsoft Office三大アプリ（Word、Excel、PowerPoint）すべてでClaude統合が完了。

**機能**:
- 長文ドキュメントのレビュー、コメントスレッド処理、修正の赤線表示
- セクション作成、条項編集
- 書式・スタイル・番号付けの保持
- **クロスアプリ共有コンテキスト**: Word・Excel・PowerPoint間で会話とデータの文脈を維持

**対象プラン**: Pro / Max / Team / Enterprise。Amazon Bedrock、Google Vertex AI、Microsoft Foundry経由の接続も可能。

**入手方法**: Microsoft AppSourceマーケットプレイス。Windows、Mac、Web版に対応。

**情報源**: [OpenPR](https://www.openpr.com/news/4468672/anthropic-rolls-out-claude-for-word-add-in-now-full-microsoft)

### 2. UK AI安全性研究所 — Mythos自律サイバー攻撃評価 🔬

**確度**: 報道事実（🔬 研究プレビューであるMythosに関する公的評価）
**日付**: 2026年4月14日

英国AI安全性研究所（AI Security Institute）がClaude Mythos Previewのサイバーセキュリティ能力を評価した結果を公開。

**主要な評価結果**:
- エキスパートレベルCTFチャレンジで**73%の成功率**
- **「The Last Ones」（TLO）32ステップ企業ネットワークシミュレーション**を完走した最初のAIモデル（10回中3回成功、平均22/32ステップ完了）
- 次点のClaude Opus 4.6は平均16ステップのみ
- 1回の評価で最大1億トークン使用
- 計算リソース増加に応じてパフォーマンスがスケール
- 「2年前はAIモデルは基本的なサイバーセキュリティ演習にすら苦戦していた」との対比

**情報源**: [Decrypt](https://decrypt.co/364141/anthropic-claude-mythos-serious-threat-overhyped-ai-security-institute) / [WLRN/NPR](https://www.wlrn.org/npr-breaking-news/2026-04-13/is-anthropics-claude-mythos-really-a-cybersecurity-bombshell)
