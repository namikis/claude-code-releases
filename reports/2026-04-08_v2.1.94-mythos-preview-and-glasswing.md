# Claude Code 調査レポート: v2.1.94・Mythos Preview・Project Glasswing

**調査日:** 2026-04-08
**対象期間:** 2026-04-07 〜 2026-04-08
**調査モード:** フルモード

---

## 概要

前回調査（2026-04-07）以降、以下の重要な動きがあった：

1. **v2.1.94**（4月7日）: Amazon Bedrock Mantle対応、デフォルト努力レベルの変更（medium→high）、`UserPromptSubmit`フック拡張、多数のバグ修正
2. **Claude Mythos Preview / Project Glasswing**（4月7日）: Anthropicの最強モデル「Mythos」がサイバーセキュリティ防御用の招待制研究プレビューとして公開。AWS・Apple・Google・Microsoft等12社が参加。一般公開は見送り
3. **Claude in Amazon Bedrock**（4月7日）: Messages APIがBedrock上で研究プレビューとして利用可能に
4. **Google/Broadcomパートナーシップ**（4月6日）: 次世代TPU数ギガワット規模の契約。年間収益ランレートが$30Bを突破
5. **大規模障害**（4月6-7日）: 2日連続でClaude.aiに障害。ログイン・音声・チャットに影響

## 主要な新機能

### v2.1.94（2026年4月7日） ✅

#### 新機能・改善

| 機能 | 詳細 |
|------|------|
| **Amazon Bedrock Mantle対応** | `CLAUDE_CODE_USE_MANTLE=1` で有効化。Bedrock powered by Mantleをサポート |
| **デフォルト努力レベル変更** | API-key、Bedrock/Vertex/Foundry、Team、Enterpriseユーザーのデフォルト努力レベルがmedium→highに変更。`/effort` で制御可能 |
| **Slack MCPヘッダー改善** | Slack MCP send-message呼び出し時にコンパクトな `Slacked #channel` ヘッダーとクリック可能なチャネルリンクを表示 |
| **`keep-coding-instructions` フロントマター** | プラグイン出力スタイルで `keep-coding-instructions` フロントマターフィールドをサポート |
| **`UserPromptSubmit` フック拡張** | `hookSpecificOutput.sessionTitle` でセッションタイトルを設定可能に |
| **プラグインスキル名改善** | `"skills": ["./"]` で宣言されたスキルがディレクトリ名ではなくフロントマターの `name` を使用 |
| **`--resume` worktree対応** | `--resume` がリポジトリのworktree間で直接動作するように改善 |
| **[VSCode] 設定パース警告** | `settings.json` のパースに失敗した場合に警告バナーを表示 |

#### 主要バグ修正

- 429レート制限レスポンスの長い `Retry-After` ヘッダーでエージェントがスタックする問題を修正
- macOSのキーチェーンがロック中にConsoleログインが無言で失敗する問題を修正
- プラグインスキルのYAMLフロントマターで定義されたフックが無視される問題を修正
- `CLAUDE_PLUGIN_ROOT` 未設定時のプラグインフックエラーを修正
- CJK・マルチバイトテキストが `stream-json` でU+FFFDに破損する問題を修正
- 長いセッションでスクロールバックに同じdiffが繰り返し表示される問題を修正
- Shift+Spaceが検索入力で "space" リテラルを挿入する問題を修正
- tmux内でハイパーリンクがブラウザタブを2つ開く問題を修正
- SDK/printモードで中断時にアシスタントの部分レスポンスが保持されない問題を修正

### Claude Mythos Preview / Project Glasswing 🔬

- **リリース状態**: 🔬 研究プレビュー（招待制、防御的サイバーセキュリティ用途限定）
- **発表日**: 2026年4月7日
- **公式ページ**: [anthropic.com/glasswing](https://www.anthropic.com/glasswing)

#### 概要

Anthropicが開発した最強のAIモデル「Claude Mythos」を、サイバーセキュリティ防御目的に限定した招待制研究プレビューとして公開。「Project Glasswing」と名付けられたこの産業横断イニシアチブには、テック大手やセキュリティ企業が参加。

#### 参加組織（12の創設パートナー）

Amazon Web Services、Anthropic、Apple、Broadcom、Cisco、CrowdStrike、Google、JPMorganChase、Linux Foundation、Microsoft、NVIDIA、Palo Alto Networks。加えて40以上の重要インフラ管理組織が拡大アクセスを取得。

#### モデル性能

| ベンチマーク | Mythos Preview | Claude Opus 4.6 |
|------------|---------------|-----------------|
| CyberGym（脆弱性再現） | **83.1%** | 66.6% |
| SWE-bench Pro | **77.8%** | 53.4% |

- 全主要OS・ブラウザで数千のゼロデイ脆弱性を発見
- OpenBSDの27年間未検出の脆弱性を発見
- FFmpegの16年間未検出の脆弱性を発見（自動ツールが500万回テストしても見逃していた）
- Linux カーネルの脆弱性を自律的に発見・チェーンして特権昇格を実現

#### 安全性への懸念と公開制限

- **一般公開は予定されていない**。Anthropicは「防御能力と同じ能力が攻撃者に武器化される可能性がある」と認識
- 政府関係者に対し「Mythosにより2026年中の大規模サイバー攻撃の可能性が大幅に高まる」と警告
- モデルが自身のセーフガードを突破する能力を持つことが判明
- 新しいセーフガードが整うまで一般リリースしない方針

#### 価格（研究プレビュー終了後）

- 入力: $25 / 100万トークン
- 出力: $125 / 100万トークン
- Claude API、Amazon Bedrock、Vertex AI、Microsoft Foundryで提供予定

#### 財政的コミットメント

- 参加組織に$100Mのモデル使用クレジット
- Linux Foundation経由でAlpha-Omega/OSSFに$2.5M寄付
- Apache Software Foundationに$1.5M寄付

### Claude in Amazon Bedrock（Messages API） 🔬

- **リリース状態**: 🔬 研究プレビュー（招待制）
- **発表日**: 2026年4月7日（APIプラットフォームリリースノート）

Anthropic公式Messages APIがAmazon Bedrock上で研究プレビューとして利用可能に。新エンドポイント `/anthropic/v1/messages` はファーストパーティClaude APIと同じリクエスト形式を使用し、AWSマネージドインフラ上でゼロオペレーターアクセスで動作。`us-east-1` で利用可能。アクセスにはAnthropicのアカウントエグゼクティブへの連絡が必要。

- **発表日**: 2026年4月6日
- Anthropicが Google と Broadcom と次世代TPU数ギガワット規模の容量契約を締結
