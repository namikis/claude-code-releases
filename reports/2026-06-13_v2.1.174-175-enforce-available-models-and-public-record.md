## 主要な新機能

### `enforceAvailableModels` 管理設定 ✅（v2.1.175）

- **リリース状態**: ✅ GA（管理設定）
- **対象**: エンタープライズ/Team 管理者（managed settings）
- **概要**: 有効化すると `availableModels` allowlist が **Default モデルにも適用**される。Default が allowlist 外のモデルに解決される場合は **allowlist 内の最初のモデルへフォールバック**。さらに**ユーザー設定・プロジェクト設定から管理側の `availableModels` リストを拡張（widen）できなくなる**。
- **意義**: これまで `availableModels` は明示選択モデルを制限していたが、Default 経由での「すり抜け」やユーザー側での拡張が可能だった。本設定でモデルガバナンスを fail-closed 化。Opus 4.8 デフォルト化・Fable 5 追加でモデル選択肢が増えた中、組織のモデル統制ニーズに対応。
- **初出**: v2.1.175 (2026-06-12)
- **情報源**: [Claude Code changelog](https://code.claude.com/docs/en/changelog)

### `wheelScrollAccelerationEnabled` 設定 ✅（v2.1.174）

- **リリース状態**: ✅ GA（設定）
- **概要**: フルスクリーン（alternate-screen）モードでのマウスホイールのスクロール加速を無効化するトグル。一定速度のスクロールを好むユーザー向け。
- **初出**: v2.1.174 (2026-06-12)
- **情報源**: [Claude Code changelog](https://code.claude.com/docs/en/changelog)

### VS Code: `/usage`（Account & usage ダイアログ）に使用量内訳追加 ✅（v2.1.174）

- **リリース状態**: ✅ GA（VS Code 拡張）
- **概要**: VS Code 拡張の Account & usage ダイアログ（`/usage`）に、**直近 24h / 7d** の使用量内訳を追加表示。**cache miss / long context / subagents / skill・agent・plugin・MCP 別**のブレークダウンを表示。CLI 側で v2.1.149 に入った per-category breakdown の VS Code 版に相当。
- **初出**: v2.1.174 (2026-06-12)
- **情報源**: [Claude Code changelog](https://code.claude.com/docs/en/changelog)

## バグ修正・パフォーマンス改善（v2.1.174 主要分）

- **`/model` ピッカーが Default の解決先モデルファミリーを隠す問題を修正** — Max/Team Premium/Enterprise では **Opus が独立行**で表示、Pro/Team では Sonnet、pay-as-you-go API アカウントでは Opus が独立行で表示されるように。
- **`/model` ピッカーのハードコードされた Sonnet バージョンラベル修正** — `ANTHROPIC_DEFAULT_SONNET_MODEL` で別 Sonnet をピン留めしているのに固定ラベルが出る問題を修正。
- **「Fable 5 is now consuming usage credits」バナーの誤表示修正** — usage ベース課金のエンタープライズアカウントで誤って表示されていた問題を修正（v2.1.173 の Fable 5 関連調整に続くフォローアップ）。
- **Bedrock GovCloud リージョン（`us-gov-*`）の推論プロファイルプレフィックス誤り修正** — `global` ではなく `us-gov` を導出するように（誤導出モデル ID で 400 エラーが発生していた）。
- **バックグラウンドセッションが別セッションの `ANTHROPIC_*` provider env を継承する問題修正** — daemon を起動したシェルから gateway URL・カスタムヘッダー・`/model` エイリアスを誤継承していた。
- **macOS/Linux でのシェルコマンド中断/kill 直後の終了時 1〜2 秒のポーズ修正**。
- **git commit co-author 表記のモデル名誤り修正**（一部モデルで誤った名前が表示）。
- **`/advisor` ダイアログが allowlist でブロック済みの保存済み advisor モデルを事前選択する問題修正**。
- **スキルの hot-reload が単一スキル変更時に全リストを再送する問題修正** — 変更されたスキルのみ再アナウンス。
- **Workflow tool の `agent()` サブエージェントが per-agent attribution ヘッダーを欠く問題修正**。
- **プリウォーム済みバックグラウンドワーカーがアイドル後に「Could not resolve authentication method」で失敗する問題修正**。

## Layer 2/3: プラットフォーム・コーポレート動向

### Anthropic Public Record（6/12 公開）📢

- **確度**: ✅ 公式（research/policy 発表、Claude Code 機能ではない）
- **概要**: Anthropic が初回 **Anthropic Public Record** の調査結果を公開。2025 年 11〜12 月に**約 52,000 人の米国民**を対象にフィールド調査。AI への期待では **48% が「がん・アルツハイマー等の疾病治療」を上位 3 つに**、次いで「障害者支援」36%。一方で**全州で最多の不安は「AI による失業」（64%）**。6/10〜11 の Economic Futures Research Fund（$200M）・Dario Amodei の段階的失業政策提言と整合する社会的文脈づくり。
- **情報源**: [Anthropic Newsroom](https://www.anthropic.com/news/anthropic-public-record)

### 6/15 サブスク課金構造変更が 2 日後に迫る 📢

- **確度**: ✅ 公式（5/14 既報、6/15 施行直前）
- **概要**: **2026-06-15 から** programmatic な Claude 利用（**Claude Agent SDK / `claude -p` CLI / Claude Code GitHub Actions / サードパーティアプリ**）が、サブスクの flat-rate プールから分離され、**ドル建ての別建て monthly credit メーター（標準 API レート課金・ロールオーバーなし）**に移行。報道される credit 額の目安は **Pro 約 $20/月・Max 5x 約 $100/月・Max 20x 約 $200/月**。**Claude アプリの対話・週次/インタラクティブ上限は変更なし**。クレジット枯渇時は overflow billing を有効にしない限り自動リクエストが停止。**Claude Code 本体の対話利用・Agent View・`claude --bg` は従来どおりサブスク内**。
- **注意**: 同日 **Claude 4（旧世代）リタイア**も一部メディアが指摘（要追跡）。
- **情報源**: [The New Stack](https://thenewstack.io/anthropic-agent-sdk-credits/) / [TechTimes](https://www.techtimes.com/articles/317625/20260602/anthropic-ends-subscription-subsidy-agents-june-15-credit-pool-replaces-flat-rate-access.htm)
