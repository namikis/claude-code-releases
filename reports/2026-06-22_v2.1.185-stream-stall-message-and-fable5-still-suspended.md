# 調査レポート: Claude Code v2.1.185 とストリーム停止メッセージ改善 / Fable 5・Mythos 5 停止継続（6/21–6/22 差分）

**調査日:** 2026-06-22
**調査モード:** フルモード（Layer 1〜3）
**対象期間:** 2026-06-20 レポート（v2.1.183）以降の差分

---

## 概要

- **Layer 1（CLI）**: **v2.1.185**（6/20 夜〜6/21）が新規リリース。内容は**ストリーム停止時のヒントメッセージ文言変更のみ**（"No response from API · Retrying in …" → "Waiting for API response · will retry in …"、トリガーを 10 秒 → **20 秒**に延長）。**新機能・新フラグ・新スラッシュコマンド・新 Hook はなし**。**v2.1.184 は欠番**（v2.1.171 / v2.1.180 / v2.1.182 に続く欠番）。
- **Layer 2（公式）**: Anthropic 公式ニュースルーム・公式リリースノート・サポートセンターのいずれも**6/17（ソウルオフィス開設）以降に新規投稿なし**。Fable 5 / Mythos 5 復旧に関する**公式発表は依然として出ていない**。
- **Layer 3（メディア）**: techjacksolutions（6/19 公開）が「**Fable 5 / Mythos 5 が本人確認＋国籍ベースのジオフェンシング付きで復旧（6/18 頃）**」と報じたが、**公式・Polymarket・公式リリースノートと矛盾**しており、**❓ 未確認（誤報・先走りの可能性が高い）**と判定。
- **Fable 5 / Mythos 5 のステータス**: **6/22 時点も停止継続**。予測市場 Polymarket は **7/1 復旧を最有力（50%）**、6/26 が 27%、6/22 が 6% と価格付け（＝6/22 時点で未復旧）。
- **課金**: 6/9〜6/22 の「Fable 5 サブスク同梱無料」期間が **6/22 で終了**し、**6/23 以降は usage credits へ移行**予定。ただし停止継続中のため事実上保留状態。

---

## 主要な変更

### 1. CLI v2.1.185 — ストリーム停止ヒントメッセージ改善 ✅ GA

- **リリース状態**: ✅ GA（npm 配布、6/20 20:59 頃 / 6/21）
- **内容**:
  - API 応答待ち中のヒント文言を **"No response from API · Retrying in …"** から **"Waiting for API response · will retry in …"** に変更（「応答なし」→「応答待ち」とニュアンスを緩和）
  - 文言が表示されるまでの**トリガー時間を 10 秒 → 20 秒に延長**（短時間の遅延で不安を煽る表示を抑制）
- **新規追加なし**: 新機能・新 CLI フラグ・新スラッシュコマンド・新 Hook イベントは**いずれもなし**
- **欠番**: **v2.1.184 は欠番**。v2.1.171 / v2.1.180 / v2.1.182 に続く 4 つ目の欠番。
- **情報源**: [GitHub Releases](https://github.com/anthropics/claude-code/releases) / [CHANGELOG.md](https://github.com/anthropics/claude-code/blob/main/CHANGELOG.md)

> 直前の v2.1.183（6/19、6/20 レポートで既報）= Auto mode の破壊的 git / IaC コマンドガードレール、`attribution.sessionUrl` 設定、`/config --help` 等。v2.1.185 はその後の純粋なメッセージング微調整リリース。

---

### 2. Fable 5 / Mythos 5 は 6/22 時点も停止継続 ⛔

- **ステータス**: ⛔ **停止継続**（6/12 の米政府輸出管理指令以降、復旧せず）
- **公式の動き**: Anthropic 公式ニュースルーム・公式リリースノート（support.claude.com / platform.claude.com）とも、**最新の関連投稿は 6/12 の停止声明**、その後の更新は 6/17 のソウルオフィス開設のみ。**復旧の公式発表は未だ無し**。
- **予測市場（Polymarket）**:
  - 「US 顧客向け復旧」市場で **7/1: 50%（最有力）、6/26: 27%、6/22: 6%** と価格付け（総出来高 約 $1.37M）
  - **6/19 / 6/22 / 6/26 のいずれの早期日付も "Yes" で未解決** ＝ 6/22 時点で未復旧
- **背景の補足**（Polymarket 記載）: 指令は「中国系グループによるモデルアクセスの疑い」＋ jailbreak 脆弱性が一部の引き金。Anthropic と商務省の交渉は継続中。
- **課金移行**: Fable 5 のサブスク同梱無料期間は **6/22 で終了**、**6/23 から usage credits 建て**に移行予定（停止中のため実質保留）。
- **情報源**: [Polymarket（Fable 5 復旧市場）](https://polymarket.com/event/claude-fable-5-restored-for-us-customers-by-20260613193753196) / [Anthropic Newsroom](https://www.anthropic.com/news) / [Claude Help Center リリースノート](https://support.claude.com/en/articles/12138966-release-notes)
