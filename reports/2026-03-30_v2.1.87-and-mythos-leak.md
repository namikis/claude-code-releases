# 調査レポート: Claude Code v2.1.87 & Claude Mythosリーク（2026年3月末）

**調査日:** 2026-03-30

## 概要

前回レポート（v2.1.86、3月28日）以降、CLIは**v2.1.87**（3月29日）がリリースされた。Cowork Dispatchのメッセージ配信に関するバグ修正のみの小規模パッチリリース。

プラットフォームレベルでは、3月26〜27日にかけて**Claude Mythos**（コードネーム "Capybara"）の情報がデータリークにより公開され、Anthropicが確認。Opus 4.6を大幅に上回る次世代モデルであり、コーディング・推論・サイバーセキュリティ能力に顕著な進歩があるとされる。正式リリース日は未発表。

API側には3月18日以降の新規リリースはなし。

---

## 詳細

### 1. Claude Code v2.1.87（2026年3月29日）

| 項目 | 説明 |
|------|------|
| **Cowork Dispatch メッセージ配信修正** | Dispatch経由で送信されたメッセージが配信されない問題を修正 |

- GitHub Release: commit `78a44f1`、作者 @ashwin-ant
- v2.1.86の2日後にリリースされたホットフィックス

### 2. Claude Mythos リーク・発表（2026年3月26〜27日）

#### 経緯

3月26日、AnthropicのCMS（コンテンツ管理システム）の設定ミスにより、約3,000件のファイルが認証なしで公開された。その中にClaude Mythosに関するドラフトブログ記事が含まれていた。Fortuneが報じ、Anthropicはモデルの存在を確認した。

#### 判明した情報

| 項目 | 詳細 |
|------|------|
| **モデル名** | Claude Mythos（コードネーム "Capybara"） |
| **位置づけ** | Opusを超える新しい最上位モデルティア |
| **リリース状態** | **発表のみ**（トレーニング完了、少数の早期アクセス顧客でパイロット中） |
| **主要能力** | コーディング、学術的推論、サイバーセキュリティでOpus 4.6を「大幅に」上回るスコア |
| **サイバーセキュリティ** | 「現在、サイバー能力において他のどのAIモデルよりもはるかに先行」との評価。脆弱性検出・攻撃手法の理解で突出 |
| **リリース戦略** | セキュリティ上の懸念からサイバー防御組織への限定早期アクセスを計画。防御側がシステムを強化する時間を確保する方針 |
| **価格** | Opusより高価な新プレミアムティアになる見込み |
